## 本リポジトリについて
ぽこピーがこれまで残した数々の名言をサウンドで再生できるサイトとなります。
作成途中となりますので、順々に対応していきます。
当方、webサイトを作るのは初めてですので、書き方がよくわかっていませんが、
温かい目で見守っていただけると幸いです。
